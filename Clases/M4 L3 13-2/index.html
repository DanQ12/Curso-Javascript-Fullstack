Acceder por ID

getElementById() selecciona un solo elemento del DOM usando su atributo id. Devuelve el elemento directamente (no una coleccion).

javascript 
                    const titulo = document.getElementById("titulo");
                    console.log(titulo.innerText)

2

Acceder por clase

getElementsByClassName() devuelve una coleccion (HTMLCollection) de todos los elementos que tienen la clase indicada. Se accede a cada unon por indice.

javascript 
                    const parrafos = document. getElementsByClassName("descripcion);
                    console.log(parrafos[0].innerText);

3

Acceder por etiquetas

getElementsByTagName() selecciona todos los elementos de un tipo especifico de etiqueta HTML y devuelve una HTMLCollection.

javascript 
                    const items = document.getElementsByTagName("li");
                    console.log(items[0].innerText);
consola

4

Acceder con querySelector

querySelector() selecciona el primer elemento que coincida con un selector CSS. Acepta selectores de clase(.), id(#), etiqueta, etc.

javascript 
                    const primerElemento = document.querySelector(".elemento");
                    console.log(primerElemento.innerText);
consola

5

Acceder con querySelectorAll

querySelectorAll() devuelve una NodeList con todos los elementos que coincidan con el selector CSS. Se puede recorrer con forEach.

javascript 
                    const elementos = document.querySelectorAll(".elemento");
                    elementos.forEach(el => console.log(el.innerText))
